# Brief: Microphone Settings Page

## Описание
Реализация страницы настроек микрофона, позволяющей пользователю выбирать устройство ввода из списка доступных. Выбор должен сохраняться и использоваться при последующих запусках записи.

## Требования
1.  **UI**:
    *   Страница `/settings` с секцией "Microphone".
    *   Отображение текущего выбранного микрофона.
    *   Кнопка "Change" для открытия модального окна выбора.
    *   Модальное окно (Dialog) со списком доступных аудио-устройств.
    *   Индикация выбранного устройства в списке.

2.  **Logic**:
    *   Получение списка устройств через `navigator.mediaDevices.enumerateDevices()`.
    *   Фильтрация по `kind === 'audioinput'`.
    *   Сохранение `deviceId` в `localStorage` (ключ `selectedMicrophoneId`).
    *   При отсутствии сохраненного `deviceId` использовать устройство по умолчанию (`default`).

3.  **Audio Capture**:
    *   Обновление `AudioConfig` в `client/src/lib/audio.ts` для поддержки `deviceId`.
    *   Использование `deviceId` при запросе `getUserMedia`.

## Критерии приемки (Definition of Done)
*   [ ] Страница настроек отображается корректно.
*   [ ] Список микрофонов загружается и отображается в модальном окне.
*   [ ] Выбор микрофона сохраняется в localStorage.
*   [ ] При перезагрузке страницы выбранный микрофон восстанавливается.
*   [ ] Запись аудио использует выбранный микрофон (проверить в коде `AudioCapture`).
