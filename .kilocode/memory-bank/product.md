# Product Context: AlfaVoice

## Цель продукта
Создание безопасного корпоративного инструмента для голосового ввода текста, который повышает производительность сотрудников, устраняя необходимость ручного набора текста, и обеспечивает конфиденциальность данных.

## Проблемы пользователей
-   Медленный набор текста вручную.
-   Необходимость переключения контекста для диктовки.
-   Риски утечки данных при использовании облачных сервисов (Google, OpenAI).
-   Отсутствие качественного распознавания русской речи в офлайн-режиме.

## Решение
Локальный (on-premise) голосовой ассистент, интегрируемый в рабочий процесс ОС.

### Ключевые возможности
1.  **Голосовой ввод везде**: Работает в Word, IDE, CRM, мессенджерах.
2.  **Умная обработка**: LLM исправляет ошибки, расставляет знаки препинания и форматирует текст.
3.  **Конфиденциальность**: Данные не покидают периметр компании.
4.  **Офлайн-режим**: Возможность работы без сети с отложенной синхронизацией.

## Пользовательский опыт (UX)
-   **Активация**: Глобальный хоткей (Ctrl+Win) или кнопка в трее/виджете.
-   **Индикация**: Ненавязчивый индикатор записи и статуса обработки.
-   **Результат**: Текст появляется в месте курсора, как если бы он был напечатан.
-   **История**: Доступ к предыдущим записям и поиск по ним.

## Целевая аудитория
-   Корпоративные пользователи (Windows Server 2019+).
-   Разработчики и инженеры.
-   Менеджеры и руководители.
