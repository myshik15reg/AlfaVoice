# Project Brief: AlfaVoice

## Обзор
**AlfaVoice** — кроссплатформенный голосовой ассистент корпоративного уровня для транскрипции речи в текст в реальном времени с последующей вставкой в активные окна приложений. Аналог Wispr Flow, но с фокусом на безопасность и работу в закрытом контуре.

## Основные цели
1.  **Транскрипция**: Высокоточная конвертация речи в текст (Whisper large-v3).
2.  **Постобработка**: Улучшение текста (пунктуация, стиль) с помощью локальных LLM (Qwen2-7B).
3.  **Интеграция**: Эмуляция ввода текста в любое активное окно ОС.
4.  **Безопасность**: Полная локальная обработка данных, отсутствие облачных зависимостей.
5.  **Офлайн-режим**: Работа без связи с сервером с последующей синхронизацией.

## Ключевые особенности
-   **Архитектура**: Клиент (Tauri/Svelte) + Сервер (Rust/Axum).
-   **Оптимизация**: CPU-only инференс (AVX-512, AMX) для Intel Xeon Sapphire Rapids.
-   **Платформы**: Windows (основная), macOS, Linux.
-   **UX**: Глобальные хоткеи, плавающая иконка, фоновая работа.
