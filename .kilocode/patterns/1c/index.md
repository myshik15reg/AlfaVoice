## Quick Reference

### jdocstring

```bsl
/// @description Описание функции
/// @param {Тип} Параметр - Описание
/// @returns {Тип} Описание
/// @task УЗ-12345
Функция МояФункция(Параметр) Экспорт
```

### Комментарии к изменениям

```bsl
// + Альфа-Лизинг. Иванов А.С. 29.11.2025. УЗ №12345.
// Описание изменений
НовыйКод();
// - Альфа-Лизинг. Иванов А.С. 29.11.2025.
```

### Именование объектов

```
Новые объекты: ал_ИмяОбъекта
Синоним: Имя Объекта (БЕЗ префикса!)
Устаревшие: Obsolete_ИмяОбъекта
```

### RabbitMQ очереди

```
queue.from.<приложение>.to.<потребитель>.<типОбъекта>.<событие>
```

### RabbitMQ routing keys

```
{типОбъекта}.{типКоманды}.{событие}
```
